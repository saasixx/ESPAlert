# ESPAlert Trademark Policy

This policy applies to the project trademarks, including the name **ESPAlert**,
the logo, icons, and any official visual identity.

## 1. Source Code vs. Trademark

- **Source code** is distributed under the **AGPL-3.0-or-later** license.
- **Trademarks** (name, logo, and visual identity) are **not** licensed under AGPL.

## 2. Permitted Use Without Prior Authorization

You may use the trademark to:

- Reference the project descriptively (e.g., in articles or comparisons).
- Indicate compatibility ("compatible with ESPAlert") without suggesting official affiliation.
- Link to the official repository.

## 3. Prohibited Use Without Written Authorization

You may not, without explicit permission from maintainers:

- Publish a copy/fork using the name **ESPAlert** as the main service or product name.
- Use the official logo or visual identity in public deployments of forks.
- Present a modified version as if it were official or endorsed by the ESPAlert team.

## 4. Forks and Redistribution

Forks and code redistribution are permitted under AGPL, but must:

- Use a distinct name and branding.
- Remove or replace official trademarks and brand assets.
- Include a clear statement that it is not the official version of ESPAlert.

## 5. Trademark Use Request

To request authorization for trademark use, open a private inquiry with the maintainers.

## 6. Rights Reserved

All trademark rights not expressly granted in this document are reserved.
