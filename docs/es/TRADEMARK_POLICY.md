# Política de Marca de ESPAlert

> 🇪🇸 **Español** | [🇬🇧 English](../TRADEMARK_POLICY.md)

Esta política aplica a las marcas del proyecto, incluyendo el nombre **ESPAlert**,
el logotipo, iconos y cualquier identidad visual oficial.

## 1. Código vs. Marca

- El **código fuente** se distribuye bajo la licencia **AGPL-3.0-or-later**.
- La **marca** (nombre, logotipo e identidad visual) **no** se licencia bajo AGPL.

## 2. Uso permitido sin autorización previa

Puedes usar la marca para:

- Referenciar el proyecto de forma descriptiva (por ejemplo, en artículos o comparativas).
- Indicar compatibilidad (“compatible con ESPAlert”) sin sugerir afiliación oficial.
- Enlazar al repositorio oficial.

## 3. Uso no permitido sin autorización escrita

No puedes, sin permiso explícito de los mantenedores:

- Publicar una copia/fork usando el nombre **ESPAlert** como nombre principal del servicio o producto.
- Usar el logotipo o identidad visual oficial en despliegues públicos de forks.
- Presentar una versión modificada como si fuera oficial o respaldada por el equipo de ESPAlert.

## 4. Forks y redistribución

Se permiten forks y redistribución del código conforme a AGPL, pero deben:

- Usar un nombre y branding distintos.
- Eliminar o reemplazar logotipos y activos de marca oficiales.
- Incluir una declaración clara de que no es la versión oficial de ESPAlert.

## 5. Solicitud de uso de marca

Para solicitar autorización de uso de marca, abre una consulta privada con los mantenedores.

## 6. Reserva de derechos

Todos los derechos de marca no concedidos expresamente en este documento quedan reservados.
